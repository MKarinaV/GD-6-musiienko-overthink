# OVERTHINK

OVERTHINK - атмосферна психологічна indie-гра, створена в Unity у жанрі one-button game.

## Основна механіка
Гравець повинен утримувати клавішу SPACE, щоб заспокоювати персонажа та не дозволити рівню стресу заповнити шкалу повністю.

## Особливості гри
- система стресу
- система енергії спокою
- динамічні негативні думки
- головне меню
- pause menu
- settings menu
- екран програшу
- збереження найкращого результату
- атмосферна аудіосистема


## Керування
SPACE - заспокоїти персонажа  
ESC - pause menu

## Мета гри
Протриматися якомога довше та не втратити себе під тиском власних думок.

## Ігровий цикл
Гравець балансує між рівнем стресу та енергією спокою.  
Якщо рівень стресу досягає максимуму - гра завершується.

## Технології
- Unity 6
- C#
- TextMeshPro
- Unity UI System
